# DayBar

macOS Menu Bar - Tracking a single anniversary.

A lightweight macOS menu bar app that tracks an important date and displays relative time-based milestones.

## Features
- Track a single important date (anniversary, birthday, or event)
- Display elapsed time (e.g. 100 days, 1 year)
- Support for milestone-based calculations
- Menu bar integration for quick access

## Tech Stack
- Swift
- SwiftUI
- macOS

## Notes
Currently supports a single event.

Future updates may include different event types such as birthdays (countdown mode) or recurring events.
